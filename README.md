# Proyecto-1-HTML-CSS---Wratta
Un personaje del juego de rol Pathfinder®, concretamente un Enano Herrero llamado Thrain "Martillo-Sordo", de las montañas de los cinco reyes, y expone ficticiamente sus creaciones y fabricaciones de herrero.

1. El "Box Model" (Modelo de Caja)

Es fundamental entender que cada elemento en HTML es una caja. Para que este portafolio tenga aire y no se vea amontonado, juego con el padding (espacio interno) y el margin (espacio externo).

2. Layout: Flexbox

Para alinear los elementos de la navegación o las tarjetas de mi proyecto, Flexbox es la herramienta más eficiente. Distribuyo el espacio de forma dinámica.

3. Tipografía y Color

Usa Google Fonts para darle personalidad.

Defino una paleta de 3 colores: uno dominante, uno secundario y uno de hover para los botones.
